# Cross - 암호화폐 간편 송금 서비스
불편한 암호화폐 송금을 간편하게 하기 위해 진행중인 암호화폐 통합지갑 프로젝트입니다.
React Native로 마이그레이션 하여 현재 저장소는 사용되지 않습니다.
